# devops-two-tier-app
## Project overview 
This document outlines the step by step process for deploying a 2 tier web application on an AWS EC2 instance.

The deployment is containerized using Docker. A full CI/CD pipeline is established using Jenkins to automate the build and deploy whenever new code is pushed to the main branch.
